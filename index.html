Podman for Debian — APT Repository

Pre-built .deb packages for Podman and its dependencies on Debian (amd64 & arm64).

Choose a Track

stable

Pinned, tested versions. Best for production and daily use.

edge

Latest upstream tags. For testing new features before they reach stable.

nightly

Built from upstream main branch HEAD daily. Bleeding-edge, may break.

Setup

Your Ubuntu version:

1. Import the signing key:

sudo mkdir -p /etc/apt/keyrings
curl -fsSL https://slazarov.github.io/podman-ubuntu/podman-ubuntu.gpg \
  | sudo tee /etc/apt/keyrings/podman-ubuntu.gpg > /dev/null

2. Add the repository — pick your track:

sudo tee /etc/apt/sources.list.d/podman-ubuntu.sources << 'EOF'
Types: deb
URIs: https://slazarov.github.io/podman-ubuntu
Suites: stable-2404
Components: main
Signed-By: /etc/apt/keyrings/podman-ubuntu.gpg
EOF
sudo tee /etc/apt/sources.list.d/podman-ubuntu.sources << 'EOF'
Types: deb
URIs: https://slazarov.github.io/podman-ubuntu
Suites: stable-2604
Components: main
Signed-By: /etc/apt/keyrings/podman-ubuntu.gpg
EOF
sudo tee /etc/apt/sources.list.d/podman-ubuntu.sources << 'EOF'
Types: deb
URIs: https://slazarov.github.io/podman-ubuntu
Suites: edge-2404
Components: main
Signed-By: /etc/apt/keyrings/podman-ubuntu.gpg
EOF
sudo tee /etc/apt/sources.list.d/podman-ubuntu.sources << 'EOF'
Types: deb
URIs: https://slazarov.github.io/podman-ubuntu
Suites: edge-2604
Components: main
Signed-By: /etc/apt/keyrings/podman-ubuntu.gpg
EOF
sudo tee /etc/apt/sources.list.d/podman-ubuntu.sources << 'EOF'
Types: deb
URIs: https://slazarov.github.io/podman-ubuntu
Suites: nightly-2404
Components: main
Signed-By: /etc/apt/keyrings/podman-ubuntu.gpg
EOF
sudo tee /etc/apt/sources.list.d/podman-ubuntu.sources << 'EOF'
Types: deb
URIs: https://slazarov.github.io/podman-ubuntu
Suites: nightly-2604
Components: main
Signed-By: /etc/apt/keyrings/podman-ubuntu.gpg
EOF

Note: The bare suite names stable, edge, and nightly are deprecated in v3.0 and will be removed in a future v3.1 release. see the migration guide →

3. Install:

sudo apt-get update
sudo apt-get install podman-suite

podman-suite is a meta-package that installs Podman and all its dependencies (crun, conmon, netavark, aardvark-dns, pasta, buildah, skopeo, and more).
