| Packages index → generated index.html | Package names/versions (nightly versions derive from upstream HEAD, attacker-influenceable) are interpolated into the HTML table |
| generated index.html → user shell | Users copy-paste setup snippets into a root shell; a snippet disabling signature verification would weaken every reader's apt trust chain |

The three-column package-versions table + `available_suites[]` accumulator loop are preserved unchanged (D-10). The NEW toggle/callout markup is static (not attacker-influenced), needs no new escaping |

No npm/pip/cargo installs in this plan — no `T-22-SC` package-legitimacy checkpoint required.
