<objective>
Create a build script for container-libs that clones the repository, builds it, and generates seccomp.json as an artifact ready for installation in Phase 12.

Purpose: container-libs (the containers-common monorepo) provides essential runtime configuration files that Podman expects. The most critical artifact is seccomp.json, which is generated via Go codegen and cannot be simply copied from upstream -- it must be built. This resolves the v1.1 tech debt where seccomp_profile references a file that does not exist on disk.

Output: A working build_container-libs.sh script wired into setup.sh, producing seccomp.json in the build directory.
</objective>
