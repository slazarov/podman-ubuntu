  <action>
Create scripts/install_container-manpages.sh following the exact project conventions (boilerplate, step_start/step_done, install -m 0644).

The script must:

1. Use the standard boilerplate (set -euo pipefail, toolpath detection with relativepath="../", source config.sh, source functions.sh, error trap).

2. Build man pages from container-libs source using go-md2man. The source .5.md files are in three locations within ${BUILD_ROOT}/container-libs:
   - common/docs/*.5.md (4 files: Containerfile.5.md, containerignore.5.md, containers-mounts.conf.5.md, containers.conf.5.md)
   - image/docs/*.5.md (10 files: containers-auth.json.5.md, containers-certs.d.5.md, containers-policy.json.5.md, containers-registries.conf.5.md, containers-registries.conf.d.5.md, containers-registries.d.5.md, containers-signature.5.md, containers-sigstore-signing-params.yaml.5.md, containers-tls-details.yaml.5.md, containers-transports.5.md)
   - storage/docs/containers-storage.conf.5.md (1 file)

3. For each .5.md file found, run: go-md2man -in "$md_file" -out "${md_file%.md}" to produce the .5 man page.

4. Use a loop pattern to iterate all three directories, converting each .5.md to .5. Use a bash function or loop like:
   ```bash
   for md_file in \
       "${BUILD_ROOT}/container-libs/common/docs/"*.5.md \
       "${BUILD_ROOT}/container-libs/image/docs/"*.5.md \
       "${BUILD_ROOT}/container-libs/storage/docs/"containers-storage.conf.5.md; do
       [[ -f "$md_file" ]] || continue
       go-md2man -in "$md_file" -out "${md_file%.md}"
       count=$((count + 1))
   done
   ```

5. Install all generated .5 files to /usr/share/man/man5/ using install -m 0644. Create the directory first with mkdir -p /usr/share/man/man5.

6. Also install the symlink from common/docs/links/.containerignore.5 if it exists:
   ```bash
   if [[ -f "${BUILD_ROOT}/container-libs/common/docs/links/.containerignore.5" ]]; then
       install -m 0644 "${BUILD_ROOT}/container-libs/common/docs/links/.containerignore.5" /usr/share/man/man5/
   fi
   ```

7. Verify installation by counting installed man pages and printing the count.

8. Do NOT install the 60+ containers-storage-*.md files from storage/docs/ that lack the .5 extension -- those are section 1 man pages for the containers-storage CLI tool which we did not build. Only install containers-storage.conf.5.

Then wire into setup.sh by adding `run_script "install_container-manpages.sh"` as the LAST line, after the existing `run_script "install_container-configs.sh"` line. Add a comment: `# Install Container Man Pages`.
