- conflicts/replaces/provides: podman
- contents: use `${DESTDIR}/usr/` as source root. Podman's `make install PREFIX=/usr DESTDIR=...` installs:
  - Binaries: usr/bin/podman, usr/bin/podman-remote, usr/bin/podmansh (if present)
  - Libexec: usr/libexec/podman/quadlet, usr/libexec/podman/rootlessport
  - Man pages: usr/share/man/man1/podman*.1 (glob via nFPM)
  - Systemd units: usr/lib/systemd/... (user and system units)
  - Completions: usr/share/bash-completion/completions/podman, usr/share/fish/vendor_completions.d/podman.fish, usr/share/zsh/vendor-completions/_podman (if present)
  - Docs: usr/share/doc/podman/ (if any)
- NOTE: The exact file list from `make install` is hard to enumerate ahead of time. Use a glob-based approach in nFPM contents where possible. For files under usr/bin/ and usr/libexec/podman/, enumerate explicitly. For man pages, use a tree-glob if nFPM supports it, or enumerate the main man pages (podman.1, podman-remote.1). The executor should run `make install PREFIX=/usr DESTDIR=/tmp/test-staging` on a real build to discover the full file tree, then populate the contents section accordingly. As a starting point, include:
  ```yaml
  contents:
    - src: "${DESTDIR}/usr/bin/podman"
      dst: /usr/bin/podman
      file_info:
        mode: 0755
    - src: "${DESTDIR}/usr/bin/podman-remote"
      dst: /usr/bin/podman-remote
      file_info:
        mode: 0755
    - src: "${DESTDIR}/usr/libexec/podman/quadlet"
      dst: /usr/libexec/podman/quadlet
      file_info:
        mode: 0755
    - src: "${DESTDIR}/usr/libexec/podman/rootlessport"
      dst: /usr/libexec/podman/rootlessport
      file_info:
        mode: 0755
  ```
  Then add glob patterns for man pages and systemd units. nFPM supports glob in src field:
  ```yaml
    - src: "${DESTDIR}/usr/share/man/man1/podman*"
      dst: /usr/share/man/man1/
      type: tree
  ```
  If `type: tree` is not available in nFPM, use explicit file listing. The executor should check nFPM docs for glob/tree support. As fallback, use a simple tree copy approach.
