**Color definitions:** RED (errors), YELLOW (warnings), GREEN (ok), NC (reset).

**Counter variables:** ERRORS=0, WARNINGS=0

**Output helpers (3 functions):**
- preflight_error(code, message, current, required, fix) - prints red [ERROR], increments ERRORS. Output to stderr.
- preflight_warn(code, message, impact, fix) - prints yellow [WARN], increments WARNINGS.
- preflight_ok(code, message) - prints green [OK].

**Individual check functions (6 functions):**

1. check_cgroups_v2() - returns 0 if cgroups v2 available:
   - Primary: check /sys/fs/cgroup/cgroup.controllers file existence
   - Fallback: findmnt -t cgroup2 /sys/fs/cgroup

2. check_subuid_configured(user) - returns 0 if user has entry in /etc/subuid:
   - grep -q "^${user}:" /etc/subuid

3. check_subgid_configured(user) - returns 0 if user has entry in /etc/subgid:
   - grep -q "^${user}:" /etc/subgid

4. check_fuse_support() - returns 0 if FUSE available:
   - Check /dev/fuse is character device (-c) AND readable (-r)

5. check_kernel_version(required) - returns 0 if current >= required:
   - Extract version: uname -r | cut -d'-' -f1
   - Compare using printf '%s\n%s\n' "$required" "$current" | sort -V | head -n1
   - Return 0 if sorted first == required (meaning current >= required)

6. check_noexec_mount(path) - returns 0 if path allows execution:
   - Get options via findmnt -T "$path" -o OPTIONS -n
   - Return 1 if options contain "noexec"

7. get_mount_info(path) - helper returning mount details for error messages:
   - findmnt -T "$path" 2>/dev/null || echo "Unknown mount"

**Main runner function - run_preflight_checks():**

Prints header "Pre-flight Validation", records start_time, runs all 5 checks:

- VAL-01: Call check_cgroups_v2. Pass -> OK. Fail -> ERROR with fix "Add systemd.unified_cgroup_hierarchy=1 to kernel cmdline and reboot".

- VAL-02: Get current_user=$(whoami). If EUID==0: OK "Running as root - subuid/subgid not required". Else check both subuid and subgid. Pass -> OK. Fail -> WARNING with fix "sudo usermod --add-subuids 100000-165535 --add-subgids 100000-165535 $user".

- VAL-03: Call check_fuse_support. Pass -> OK. Fail -> ERROR with fix "Install fuse3 package or enable FUSE in kernel config".

- VAL-04: Get kernel_version from uname. Check >= 5.11 first (OK). If not, check >= 4.18 (WARNING "below recommended 5.11"). If neither, ERROR "below minimum 4.18".

- VAL-05: Loop over /tmp and ${HOME:-/home}. Also include TMPDIR if set. Check each with check_noexec_mount. Collect failing paths in noexec_paths array. None failed -> OK. Any failed -> ERROR with fix about /etc/fstab or TMPDIR.

Print footer with timing (end_time - start_time in seconds), error count, warning count.

Exit logic: return 1 if ERRORS > 0. Print "Warnings detected" message if WARNINGS > 0. return 0 otherwise.

**Standalone execution block at end of file:**
```bash
if [[ "${BASH_SOURCE[0]}" == "${0}" ]]; then
    run_preflight_checks
    exit $?
fi
```
