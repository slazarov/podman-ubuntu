<objective>
Persist Go build cache (GOCACHE) and module cache (GOMODCACHE) across all Go component builds so that shared dependencies are compiled/downloaded once and reused.

Purpose: Go components (podman, buildah, skopeo) share ~80% of their Go module graph. Currently each build uses /tmp/go-build (ephemeral) or mktemp (discarded). Persisting the cache to /var/cache/go-build and /var/cache/go-mod gives 20x faster rebuilds on warm cache.

Output: All 5 Go build scripts use a persistent shared GOCACHE/GOMODCACHE directory. Uninstall cleans up these directories.
</objective>
